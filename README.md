# 🛒 Smart Grocery List & Inventory Manager

A production-inspired MERN Stack application that helps users efficiently manage household and business grocery inventory through real-time stock tracking, expiry monitoring, automated shopping list generation, and intelligent low-stock alerts.

Designed as a Full Stack Development portfolio project, this application demonstrates modern software engineering practices including secure authentication, RESTful API architecture, database modeling, inventory management workflows, dashboard analytics, and responsive user interface development.

---

## 📌 Project Overview

Managing groceries manually often leads to:

* Forgotten purchases
* Duplicate buying
* Expired food items
* Inventory shortages
* Increased food waste

The Smart Grocery List & Inventory Manager solves these challenges by providing a centralized platform where users can monitor stock levels, track expiry dates, receive alerts, and automatically generate shopping lists based on inventory conditions.
